# 💼 Personal Portfolio Website

A modern, responsive and professional portfolio website built to showcase skills, projects and services with a clean UI and smooth user experience.

---

## 🚀 Overview

This portfolio website is designed to represent a full stack developer in a premium and modern way. The focus was on performance, responsiveness, and clean design aesthetics.

The layout is fully responsive across all devices, ensuring a seamless experience on mobile, tablet, and desktop.

---

## 🎯 Purpose

* Showcase developer skills and technologies
* Highlight real-world projects
* Present services in a professional way
* Provide an easy way for clients to get in touch
* Build a strong personal brand online

---

## 🛠️ Technologies Used

### Frontend

* React.js
* Tailwind CSS
* Framer Motion

### Icons & UI

* React Icons
* Custom gradient UI design
* Glassmorphism effects (backdrop blur, soft shadows)

### Email Integration

* EmailJS (for contact form functionality)

### Notifications

* React Toastify (for success/error messages)

---

## 🎨 Design Features

* Modern gradient backgrounds
* Clean typography with proper hierarchy
* Glassmorphism cards (blur + transparency)
* Smooth hover effects and transitions
* Micro animations using Framer Motion
* Balanced spacing and layout

---

## 📱 Responsive Design

The website is fully responsive and optimized for all screen sizes:

* **Mobile:**

  * Stacked layout
  * Center aligned content
  * Optimized spacing (no extra empty space)

* **Tablet:**

  * Improved spacing and readability
  * Balanced grid layouts

* **Desktop:**

  * Two-column layouts (image + text)
  * Full-width container (`max-w-7xl`)
  * Premium spacing and alignment
